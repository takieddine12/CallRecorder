Selecting recordings

While tapping a recording opens the audio player, long pressing a recording selects it. When a recording is selected, the contact details screen enters theselect mode. When the select mode is on the following things happen:

  1. The recordings are decorated with a checkbox at the left extremity. The selected recordings have this checkbox checked.
  2. The top bar hides the buttons and the menu for managing the contact and shows instead three buttons and a menu for managing the recording.
  3. While the select mode is on, simply tapping a recordings selects it.
The select mode is cleared when there are no more selected recordings or when one or more recordings are deleted.

Select mode actions

While in select mode you can manage the selected recordings in several ways:

Tapping the button will open a dialog which shows some useful information about the currently selected recording, like date and hour of the call, its duration, the file size, the audio format and the location of the audio file. If two or more recordings are selected, this dialog only shows the total file size of the selected recordings.

Tapping the button will select all of the recordings.

Tapping the button will show a menu with 2 options:

  • Move to private storage
  • Move to public storage
If at least one of the selected recordings is already located in the private space of the application, the first option will be disabled. If you tap the second option a folder chooser dialog will show up assisting you in choosing the folder where you want to move the selected recordings. If you move the recordings in a public storage location you will be able to copy them to a computer or another mobile device.
After the move finishes, a final dialog will inform you if the move was done successfully or if errors have happened during the file move.

Tapping the button in the right end of the top bar will show a menu with two entries: "Rename" and "Delete".
"Rename" allows you to give a name of your choice to the recordings. If more than one recording is selected this entry will be disabled.
"Delete" will delete the selected recordings. Caution: this action cannot be undone.

Unavailable audio files

If a recording is stored in a public storage you can accidentally delete or move the audio file. When this happens the recording will be grayed out and decorated with an exclamation sign: If you cannot recover the associated audio file you should delete the recording.
